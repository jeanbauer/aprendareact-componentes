# Banner
São componentes utilizados para informar a usuária em qual página ela se encontra ou destacar algum conteúdo.


### 1. Banner com dados
É a única opção de uso do banner de forma opinionada, ou seja, um componente que já contém as regras de estilização necessárias para montar um banner no estilo para o Aprenda React (<a target="_blank" href="https://www.aprendareact.com.br/perfil">exemplo</a>).
Para visualizar este banner com as cores do tema basta não enviar a prop `colors`.
